# KAT-cli
KickassTorrents cli

    Usage:
      kat.py <search> [-c <category>] [-t]
      kat.py (-h | --help)
      kat.py --version

    Options:
      -h --help                 Show this screen.
      --version                 Show version.
      -c <category>             Category to search.
      -t                        Launch Torrent.
